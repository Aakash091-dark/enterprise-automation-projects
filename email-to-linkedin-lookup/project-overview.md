# Email-to-LinkedIn Lookup Microservice

## 📌 Overview

A scalable FastAPI-based microservice designed to map email addresses to LinkedIn profiles using structured scraping and database-backed validation.

> Note: Source code excluded due to confidentiality agreements.

---

## 🎯 Problem Statement

Organizations required a scalable system to connect email identifiers with LinkedIn profile intelligence.

The system needed to:

- Handle structured inputs
- Connect with cloud database
- Provide API-based access
- Maintain reliability across deployments

---

## 🏗 Architecture Approach

Client Request
→ FastAPI Service
→ Input Validation
→ LinkedIn Scraper
→ Data Refinement
→ RDS Database
→ JSON API Response

---

## 🛠 Tech Stack

- Python
- FastAPI
- RDS (Cloud relational database)
- GitHub Actions (CI/CD)
