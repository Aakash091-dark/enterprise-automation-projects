# Orbit Script – Email Scraper & Lead Automation System

## 📌 Overview

Orbit Script is a production-level lead automation system designed to extract, validate, and process LinkedIn-based contact data. The system integrates asynchronous scraping, structured validation pipelines, and automated reporting mechanisms.

> Note: Source code excluded due to confidentiality agreements.

---

## 🎯 Problem Statement

Manual lead collection and validation is time-consuming and error-prone.The goal was to design an automated pipeline capable of:

- Extracting LinkedIn-based data
- Validating structured inputs
- Removing duplicates
- Generating reliable reports

---

## 🏗 Architecture Approach

User Input
→ Async Scraper (Playwright)
→ Data Processor
→ Duplicate URL Checker
→ JSON/CSV Formatter
→ Reporting System

The architecture follows modular separation between scraping, validation, and reporting layers.

---

## 🛠 Tech Stack

- Python
- Playwright (Async scraping)

- JSON & CSV processing
